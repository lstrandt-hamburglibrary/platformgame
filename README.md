# Pharaoh's Curse

A modern remake of the classic 1983 Atari game "The Pharaoh's Curse" by Steve Coleman.

## About

Explore 16 interconnected rooms of an ancient Egyptian pyramid, collecting treasures while avoiding the Pharaoh, his mummy guards, and deadly traps. Watch out for the mysterious winged avatar that can teleport you to random rooms!

## Game Features

- **16 interconnected rooms** forming a complete pyramid tomb
- **Rope climbing** to navigate vertical shafts
- **Key & door system** to unlock new areas
- **Enemies:** Mummy guards and the vengeful Pharaoh
- **Winged Avatar** randomly teleports you and steals your keys
- **Trap triggers** more dangerous than enemies
- **Treasure collection** for extra lives
- **Unique running mechanic** - must run before jumping

## Controls

- **Arrow Keys** or **WASD** - Move left/right
- **Up/W** while on rope - Climb up
- **Down/S** while on rope - Climb down
- **Space** - Jump (must be running first!)
- **Shift** - Shoot
- **R** - Restart from beginning

## Technology

- Built with Phaser 3 game engine
- Progressive Web App (PWA) - installable and works offline
- Text-based level design for easy editing

## Development

This is a tribute to the original 1983 game published by Synapse Software.

Original game design by Steve Coleman.
Modern remake created 2025.

## How to Play

1. Explore the pyramid rooms
2. Find keys to unlock doors
3. Collect all treasures (scarabs, gold, jewels)
4. Avoid or defeat the Pharaoh and mummies
5. Complete all 16 rooms to win!
